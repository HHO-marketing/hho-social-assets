# hho-social-assets

Öffentliche, **bereits menschlich freigegebene** Social-Media-Medien der HHO Marketing.

## Zweck

Dieses Repository stellt ausschließlich Bilddateien öffentlich über stabile
Roh-URLs bereit, die für die Veröffentlichung auf den HHO-Social-Media-Kanälen
(Instagram, Facebook) freigegeben wurden.

## Regeln

- **Nur freigegebene Medien.** Hier liegen ausschließlich menschlich freigegebene,
  zur Veröffentlichung bestimmte Motive – keine internen Entwürfe.
- **Keine unfertigen Varianten.** Zwischenstände, Roh- oder Alternativvarianten
  gehören nicht in dieses Repository.
- **Keine Secrets.** Keine Tokens, Zugangsdaten oder `.env`-Inhalte.
- **Stabile URLs – keine stillschweigende Ersetzung.** Bereits veröffentlichte
  Dateien dürfen nicht überschrieben oder ersetzt werden, weil externe Beiträge
  auf ihre unveränderliche URL verweisen. Änderungen erfolgen nur als **neue**
  Datei.

## Struktur

```
posts/
  <YYYY-MM-DD>/
    <content-id>-<variante>.jpg
```
